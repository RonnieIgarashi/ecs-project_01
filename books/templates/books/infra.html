Infrastructure — ECS × Terraform

AWS ECS（Fargate）+ Terraform によるコンテナインフラ構成図

使用技術
レイヤー 技術 概要
IaC Terraform 全AWSリソースをコードで管理（terraform apply / destroy）
コンテナ基盤 AWS ECS (Fargate) サーバーレス起動。Auto Scalingによるタスク増減
イメージ管理 Amazon ECR DockerイメージのプライベートレジストリをTerraformで構築
ロードバランサー ALB Application Load Balancer。ECSタスクへのルーティング
データベース Amazon Aurora マネージドRDB。マルチAZへの自動バックアップ対応
ログ管理 CloudWatch Logs コンテナログを自動収集。保持期間はTerraformで定義
アプリケーション Django (Python) 蔵書管理WebアプリをDockerコンテナとして稼働
Terraform構成
terraform/ ├── main.tf # VPC・サブネット（data source） ├── variables.tf # project_name / aws_region / vpc_id / subnet_ids ├── iam.tf # ECSタスク実行ロール ├── ecr.tf # ECRリポジトリ ├── ecs.tf # ECSクラスタ・タスク定義・セキュリティグループ・サービス └── cloudwatch.tf # ロググループ（保持期間 7日）
